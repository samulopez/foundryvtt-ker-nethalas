
# Ker Nethalas: Into the Midnight Throne for Foundry VTT

**foundryvtt-ker-nethalas** is an unofficial system for playing [Ker Nethalas: Into the Midnight Throne](https://blackoathgames.com/ker-nethalas-into-the-midnight-throne) on [Foundry VTT](http://foundryvtt.com/)

## Features

- Character sheets with attributes, skills, inventory and mechanics
- Monster sheets with stats, traits and actions
- Minion sheets
- Rollable tables

### Content

- Spoils and loot
- Overseers
- Monsters
- Minions

## Installation

Install the system from within Foundry (Configuration and Setup, Game Systems tab, Install System button). Find package Ker Nethalas and install.

Alternatively, you can manually install the system by following these steps:

1. Inside Foundry, select the Game Systems tab in the Configuration and Setup menu.
2. Click the Install System button and enter in the Manifest URL the following URL: `https://github.com/samulopez/foundryvtt-ker-nethalas/releases/latest/download/system.json`

3. Click Install and wait for installation to complete.
